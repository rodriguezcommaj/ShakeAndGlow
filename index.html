Shake and Glow

Shake and Glow is a super simple, really useless exercise to get mess around with the CSS3 text-shadow, transition, and keyframe animation properties. Only bothered testing it in the most recent versions of Firefox and Chrome, so don't use this in production, or else you're likely to get laughed at by your QA team. Hell, I didn't even bother with the other vendor prefixes.

Pretty simple stuff going on - text-shadows create the glow effect with multiple shadows for the element. I used rgba for the color so that I could control opacity. It uses white for the shadow, with lowered opacity so that it looks nice on pretty much any background color. The shake part uses a very quick, infinitely repeating keyframe animation to change the font-size and make it induce seizures.
